# Replace String App
+ An Console App that replaces some string from App.config with a conditional that the **full file path** does not contains the following word:
  + "-cli-"
  + "\obj\"
  + "\bin\"
  + "\release\"
  + "\debug\"
  + "\.history\"
  + "_components\"
  + "_modules\"
  + "\dist\"
  + "\test\"
  + "\tests\"
  + "\vendor\"
